# User Service API

Сервис управления пользователями для тестового задания Effective Mobile.


## 📋 Требования

- ✅ Регистрация пользователя
- ✅ Авторизация (JWT)
- ✅ Получение пользователя по ID (себя или админ)
- ✅ Список пользователей (только админ)
- ✅ Блокировка пользователя (себя или админ)
